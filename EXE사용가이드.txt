============================================================
AI 글쓰기 자동화봇 EXE 파일 사용 가이드
============================================================

【 생성된 파일 위치 】

📁 dist 폴더 안에 있습니다:
   dist\AI글쓰기자동화봇.exe


【 실행 방법 】

1. dist 폴더로 이동
2. "AI글쓰기자동화봇.exe" 더블클릭
3. 프로그램 자동 실행!


【 중요 사항 】

⚠️ 첫 실행 시:
   - Windows Defender가 차단할 수 있습니다
   - "추가 정보" → "실행" 클릭하면 실행됩니다
   - 안전한 프로그램이니 걱정 마세요!

⚠️ 필요한 것:
   ✅ Chrome 브라우저 (필수!)
   ✅ 인터넷 연결 (Gemini API 통신)
   ❌ Python 설치 불필요! (exe에 포함됨)


【 다른 PC로 옮기기 】

✅ 단일 파일이므로 그냥 복사하면 됩니다!

1. "AI글쓰기자동화봇.exe" 파일만 복사
2. USB나 이메일로 전송
3. 다른 PC에서 실행
4. 끝!

※ Python, 라이브러리 설치 필요 없음!


【 파일 크기 】

약 100-150MB (모든 라이브러리 포함)


【 실행 속도 】

- 첫 실행: 5-10초 (압축 해제 중)
- 이후 실행: 3-5초


【 문제 해결 】

Q: 실행이 안 돼요
A: Chrome 브라우저가 설치되어 있는지 확인하세요

Q: Windows Defender가 차단해요
A: "추가 정보" → "실행" 클릭

Q: 느려요
A: 첫 실행은 느립니다. 정상입니다!

Q: config.json이 어디 생기나요?
A: exe 파일과 같은 폴더에 자동 생성됩니다


【 배포 방법 】

✅ 회사/친구에게 공유:
   - exe 파일만 보내면 끝!
   - Python 몰라도 사용 가능!

✅ USB로 들고 다니기:
   - USB에 복사해서 어디서든 실행!
